Detailed Description

A class used to improve mesh connectivity.

The edges are flipped in order to reduce the number of edges connected to a node The optimal number of edges to a node is six. If we additionally set triangulateFacesto true, that results in triangles of 60Â° in each angle and therefore a nearly ideal mesh. An additional option projectToLandBoundary defines whether we want to project to the land boundary.
